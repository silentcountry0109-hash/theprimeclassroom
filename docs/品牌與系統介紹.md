# The Prime 質數教室 — 品牌與系統介紹

**國小數學個別指導的領導品牌**

---

## 關於質數教室

質數教室是專為台灣國小學生設計的數學個別指導品牌。我們相信每個孩子都有獨特的學習節奏，因此採用「一對一至一對五」的小班制模式，讓每位學生都能獲得老師的專注引導，真正理解數學而非死記公式。

我們的名字「質數」，正如數學中只能被 1 和自己整除的質數一樣，象徵著每個孩子獨一無二的學習潛能。

---

## 品牌核心理念

### 個別指導，因材施教
每位學生擁有獨立的學習進度與教材，老師依據學生當下的理解程度即時調整教學方式，不讓任何一個孩子在團體中被忽略。

### 理解優先，拒絕填鴨
我們強調「認知轉譯」——將抽象的數學概念轉化為孩子能理解的具體經驗，建立真正的數學思維，而非短暫的考試記憶。

### 透明學習歷程
每堂課後，老師都會撰寫詳細的聯絡簿，記錄孩子的學習單元、課堂表現、小考成績和回家作業，家長隨時掌握孩子的學習狀況。

### 溫暖安心的學習環境
日式簡約的教室設計，營造溫馨專注的學習氛圍。讓孩子在安心的環境中，享受探索數學的樂趣。

---

## 教學特色

### 自編教材體系
質數教室擁有完整的一至六年級自編教材（共 12 冊），每冊對應一個學期的學習內容。教材結構清晰：

**觀念講解 → 範例演練 → 隨堂練習 → 課後習題**

教材內容緊扣國小數學課綱，同時融入數學思維訓練，幫助孩子從「學校數學」進階到「數學思考力」。

### 三階段學習地圖
- **基礎鞏固（Essential）**：對接學校進度，補齊觀念缺口
- **進階提升（Advanced）**：培養邏輯推理與解題策略
- **菁英挑戰（Elite）**：資優思維訓練與數學探索

### 認證師資
所有老師均須通過總部嚴格的培訓與考核，涵蓋教學方法、兒童心理學、認知轉譯技巧等專業領域，取得「質數教室認證老師」資格後方可授課。

## 數位平台功能

質數教室打造了完整的線上管理平台，讓教學管理、預約排課、學習追蹤都能數位化完成。

### 家長專屬功能

| 功能 | 說明 |
|------|------|
| 智慧搜尋 | 依縣市、區域搜尋附近教室，查看教室環境照片、老師介紹與評分 |
| 線上預約 | 即時查看可用時段，選擇老師與時間，一鍵預約。支援「週期性排課」，一次預約數週同時段課程 |
| 收藏教室 | 將喜愛的教室加入收藏，下次搜尋時自動置頂 |
| 孩子管理 | 建立多位孩子的學習檔案，記錄年級、學校等基本資訊 |
| 聯絡簿 | 課後即時查看老師撰寫的聯絡簿，掌握孩子的學習單元、課堂表現、小考成績與作業 |
| 預約管理 | 查看所有預約紀錄，支援取消（開課前 4 小時）與行事曆匯出（.ics 同步手機） |
| 教材商城 | 線上選購質數教室自編教材與學習輔具 |

### 老師專屬功能

| 功能 | 說明 |
|------|------|
| 月行事曆 | 一目瞭然的月曆檢視，標記有課的日期，點擊展開當日學生名單與座位狀況 |
| 聯絡簿撰寫 | 課後為每位學生記錄上課內容、學習表現、小考成績、作業與備註，一鍵送達家長 |
| 學生歷程 | 查看每位學生的歷次聯絡簿，追蹤長期進步軌跡，調整教學策略 |
| 個人資料 | 維護教學簡介與專長標籤 |

### 分校管理功能

| 功能 | 說明 |
|------|------|
| 營運數據 | 即時掌握預約數、滿班率、每日使用率圖表、老師個別績效 |
| 排課管理 | 建立每週課表、指派老師、設定座位上限 |
| 師資管理 | 新增老師、建立登入帳號、啟用/停用（不需刪除帳號）、認證管理 |
| 分校資訊 | 編輯教室介紹、特色標籤、鄰近學校，提升搜尋能見度 |
| 預約管理 | 查看所有學生預約記錄與上課狀態 |

### 總部管理功能

| 功能 | 說明 |
|------|------|
| 全國營運總覽 | 一頁掌握所有分校的學生數、老師數、預約量、滿班率，支援篩選與排序 |
| 加盟管理 | 開設新分校、指派主任帳號、啟用/停用分校 |
| 官網 CMS | 不需工程師即可修改首頁所有文案，即時上線 |
| 內容管理 | 管理 FAQ、成功案例、系統公告 |
| 商城營運 | 商品上架、庫存管理、訂單處理 |
| 帳號管理 | 全系統使用者與角色權限管理 |

## 品牌官網

質數教室官網採用日式簡約設計風格，以蒂芬妮藍（Tiffany Blue）為主色調，搭配珊瑚粉與暖黃色系，傳遞溫暖專業的品牌形象。

### 官網內容

- **品牌理念**：完整傳達質數教室的教育哲學
- **教學方法**：圖解說明個別指導的教學流程
- **學習地圖**：展示三階段學習進程
- **教材展示**：12 冊教材目錄，可互動查看各冊單元內容
- **推薦老師**：老師介紹輪播，展示專長與評價
- **家長見證**：真實成功案例分享
- **常見問題**：分類 FAQ 解答家長疑慮
- **教室搜尋**：即時搜尋全台教室，查看詳情與可用時段

### 總部提供

- 認證師資培訓課程
- 全功能數位管理平台
- 營運數據分析工具
- 線上預約與家長溝通系統

### 加盟夥伴負責

- 日常教學營運
- 在地招生與家長服務
