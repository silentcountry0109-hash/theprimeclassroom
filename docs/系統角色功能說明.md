# The Prime 質數教室 — 系統角色功能與使用場景說明

---

## 一、總部管理員（HQ Admin）

**登入方式：** `/hq-login`（帳號密碼登入）

### 功能總覽

| 功能模組 | 說明 |
|---------|------|
| 營運總覽 | 即時顯示全國學生數、老師數、分校數、預約數、平均滿班率、本月預約量；下方為各分校營運分析卡片，可依縣市篩選、依預約數/滿班率/學生數排序 |
| 常見問題管理 | 新增、編輯、刪除官網 FAQ，可設定分類與排序 |
| 成功案例管理 | 管理家長見證（學生姓名、年級、家長姓名、見證內容、標籤） |
| 加盟分校管理 | 新增/編輯分校資料（名稱、地址、電話、描述、照片）、啟用/停用分校、為分校建立主任帳號 |
| 老師管理 | 總部層級的老師資料管理（姓名、簡介、專長、認證狀態、評分） |
| 時段管理 | 依分校篩選，新增排課時段（日期、時間、指派老師、座位數），監控預約狀況 |
| 帳號管理 | 查看所有註冊使用者、變更角色（管理員/分校主任/家長）、為使用者建立帳號密碼 |
| 公告管理 | 發布、編輯、刪除系統公告 |
| 商城管理 | 商品上架（名稱、分類、價格、折扣價、庫存、圖片）、訂單管理（查看明細、更新狀態） |
| 官網編輯 | CMS 編輯器：修改首頁各區塊文案（Hero 標語、品牌理念、教學特色、流程步驟、CTA、頁尾資訊），即時更新無需改程式碼 |

### 使用場景

- 每日查看營運總覽，掌握各分校表現與趨勢
- 新分校開幕時，建立分校資料並指派主任帳號
- 定期更新官網 FAQ、成功案例與公告
- 管理教材商城的商品上下架與訂單出貨
- 行銷活動時修改官網 Hero 區文案與 CTA

## 二、分校主任（Franchise Admin）

**登入方式：** `/franchise-login`（帳號密碼登入）

### 功能總覽

| 功能模組 | 說明 |
|---------|------|
| 分校總覽 | 顯示本分校老師數、可用時段數、總預約數、已確認預約數；可選擇日期區間（本週/兩週/月/季/自訂）查看滿班率、每日使用率長條圖、各老師績效表 |
| 分校資訊 | 編輯分校介紹、電話、特色標籤（如「小班制」「專業師資」）、鄰近學校列表 |
| 師資管理 | 新增/編輯老師資料、切換認證狀態、啟用/停用老師（不需刪除帳號）、為老師建立登入帳號、重設老師密碼 |
| 時段管理 | 建立排課時段（日期、起迄時間、指派老師、座位上限）、刪除空時段、查看各時段預約狀況 |
| 預約管理 | 查看所有學生預約記錄（學生姓名、年級、學校、日期時間、狀態） |

### 使用場景

- 每週排定下週課表，指派老師到各時段
- 新老師入職時建立資料和登入帳號
- 老師請長假時停用帳號，復職後重新啟用
- 查看分校營運數據，分析滿班率和老師績效
- 更新分校介紹和特色標籤以吸引家長

## 三、老師（Coach）

**登入方式：** `/coach-login`（帳號密碼登入，由分校主任建立帳號）

### 功能總覽

| 功能模組 | 說明 |
|---------|------|
| 行事曆 | 月曆檢視，標記有排課的日期；點擊日期展開當日時段清單，每個時段顯示時間、教室名稱、已預約學生名單（姓名、年級）、座位使用狀況 |
| 聯絡簿撰寫 | 針對每個時段的學生批次撰寫聯絡簿：共用資訊（上課單元、教學進度）+ 個別學生（學習表現、課堂筆記、小考成績、回家作業、老師備註給家長） |
| 我的學生 | 查看所有曾上過課的學生列表；點入可瀏覽該學生的歷次聯絡簿（學習歷程），追蹤長期進步軌跡 |
| 個人資料 | 查看/編輯自我介紹與教學專長 |

### 使用場景

- 每天上課前查看行事曆，確認今日學生名單
- 課後立即填寫聯絡簿，記錄每位學生的學習狀況和成績
- 學期中檢視特定學生的學習歷程，調整教學策略
- 定期更新個人簡介和專長標籤

## 四、家長（Parent）

**登入方式：** `/parent-login`（帳號密碼登入）

### 功能總覽

| 功能模組 | 說明 |
|---------|------|
| 首頁總覽 | 顯示最近一堂課倒數計時、各孩子的待上課/已完成堂數摘要 |
| 預約課程 | 依縣市/區域搜尋教室 → 查看教室詳情（環境照片、老師介紹、評分）→ 選擇日期和時段 → 選擇孩子 → 確認預約。支援「週期性排課」一次預約未來數週同時段 |
| 收藏教室 | 在教室搜尋列表點擊愛心收藏，收藏的教室會自動置頂顯示 |
| 我的孩子 | 新增/編輯孩子資料（姓名、性別、年級、學校、生日、健康/學習備註） |
| 預約紀錄 | 查看所有預約歷史，可取消未來課程（需於開課前 4 小時操作）、匯出行事曆檔案（.ics）同步到手機 |
| 聯絡簿 | 查看每個孩子的所有聯絡簿，依日期排序。可篩選特定孩子。每張聯絡簿完整顯示上課單元、進度、學習表現、小考成績、作業、老師備註 |
| 商城 | 瀏覽教材商品、加入購物車、結帳下單、查看訂單紀錄 |

### 使用場景

- 首次使用：註冊帳號 → 新增孩子資料 → 搜尋住家附近教室 → 預約試聽
- 每週固定：使用週期性排課功能一次預約 4 週課程
- 課後追蹤：查看老師撰寫的聯絡簿，了解孩子課堂表現和作業
- 收藏常去的教室，下次預約時快速找到
- 需要教材時到商城選購

## 五、官網訪客（未登入）

**瀏覽路徑：** 首頁 `/`、教室搜尋結果 `/search`、教室詳情 `/classroom/:id`

### 可使用功能

| 功能 | 說明 |
|------|------|
| 品牌官網 | 瀏覽品牌理念、教學方法、學習地圖、教材內容、核心特色 |
| 老師輪播 | 查看推薦老師的介紹卡片 |
| 家長見證 | 閱讀成功案例輪播 |
| FAQ | 瀏覽常見問題 |
| 教室搜尋 | 依地區、星期、時段搜尋可用教室，查看教室詳情頁（環境照片、老師資料、時段資訊） |
| 註冊/登入 | 進入家長登入或各角色登入頁面 |
