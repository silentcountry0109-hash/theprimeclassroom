## 一、登入系統（主任端）

### 登入網址

開啟瀏覽器，進入分校後台登入頁面。

> **網址**：系統網址後面加上 `/franchise-login`
> 例如：`https://your-domain.com/franchise-login`

### 步驟

1. 進入登入頁面後，確認上方已切換到 **「分校主任」** 頁籤
2. 輸入您的帳號（由總部提供）
3. 輸入密碼
4. 點擊 **「登入管理系統」** 按鈕

![分校主任登入頁](step1-dashboard.png)

> 💡 **提示**：帳號和初始密碼由總部設定，請妥善保管。若忘記密碼請聯繫總部。

### 登入頁面說明

| 項目 | 說明 |
|------|------|
| 分校主任頁籤 | 主任使用此頁籤登入後台管理系統 |
| 老師頁籤 | 老師使用此頁籤登入老師工作介面（詳見第二部分） |
| 帳號格式 | 由總部分配，通常為中文名稱或英文代號 |

## 二、儀表板總覽

登入後，您會看到 **分校總覽** 頁面，這是您每次登入的首頁。

### 今日數據卡片

頁面頂端顯示四個即時數字：

| 卡片 | 說明 |
|------|------|
| 👩‍🏫 在職老師數 | 目前分校的活躍老師人數 |
| 📅 今日時段 | 今天安排的上課時段數 |
| 📋 今日預約 | 今天已預約的學生人次 |
| ✅ 今日出席 | 今天已標記到場的學生人次 |

### 今日上課一覽

儀表板中央顯示 **「今日上課」** 列表，清楚呈現每個時段的：
- 上課時間
- 指派老師
- 預約人數 / 上限人數
- 學生姓名列表

### 側邊導覽選單

左側選單包含所有功能入口：

```
📊 分校總覽
🏫 分校資訊
👩‍🏫 師資管理
📅 時段管理
📋 預約管理
📒 聯絡本管理
```

## 三、分校資訊設定

點選左側 **「分校資訊」**，進入分校基本資訊管理頁面。

### 查看模式

進入頁面後，預設為 **查看模式**，顯示：
- 分校名稱、地址、電話
- 分校介紹文字
- 特色標籤（例如：家長好評推薦、年度績優校區）
- 鄰近學校列表
- 營業時間表

### 進入編輯模式

點擊右上角的 **「✏️ 編輯資訊」** 按鈕，進入編輯模式。

---

### 3-1 基本資訊編輯

| 欄位 | 說明 |
|------|------|
| 電話 | 分校聯絡電話，格式如 02-xxxx-xxxx |
| 分校介紹 | 顯示在家長端的分校介紹文字 |

---

### 3-2 ✨ AI 自動生成分校介紹

這是一個特別實用的功能！系統可以根據您的分校資料，自動生成一段專業的分校介紹文案。

**操作步驟：**

1. 在編輯模式下，找到「分校介紹」欄位
2. 點擊欄位右上角的 **「✨ AI 自動生成」** 按鈕
3. 系統會顯示「生成中…」（約 3-5 秒）
4. AI 自動填入一段 80-120 字的繁體中文介紹文案
5. 您可以直接使用，或進行修改後再儲存

> 💡 **提示**：AI 會參考您的分校名稱、地區、鄰近學校和標籤來生成最適合的文案，內容親切自然，非常適合展示給家長看。

---

### 3-3 標籤設定

從下拉選單選擇適合分校的特色標籤：

- 家長好評推薦
- 年度績優校區
- 成績進步快速
- 新開幕
- 小班教學

> 可以選擇多個標籤，點選標籤旁的 ✕ 可以移除。

---

### 3-4 鄰近學校設定

輸入鄰近的國小名稱，讓家長更容易搜尋到您的分校：

1. 在輸入框輸入學校名稱（例如：大安國小）
2. 按 **Enter** 或點擊 **「加入」** 按鈕
3. 學校名稱會顯示為標籤，點 ✕ 可移除

---

### 3-5 營業時間設定

設定每週各天的上下課時間：

1. 勾選每天是否營業（開關切換）
2. 設定開始時間與結束時間
3. 休息日（例如週日）請關閉開關

**特殊節日設定（假日調整）：**

針對特定日期（如連假、颱風天）可以單獨設定：

1. 在「特殊節日」區域輸入日期（格式：YYYY-MM-DD，例如 2026-06-07）
2. 選擇該日是否營業
3. 若營業，設定特殊時段的開始/結束時間
4. 可加入備註（例如：端午節）
5. 點擊 **「加入」** 按鈕

> ⚠️ **注意**：特殊節日設定會覆蓋一般營業時間，且在批次建立時段時，系統會自動略過標記為「休息」的特殊日期。

---

### 儲存變更

所有欄位編輯完成後，點擊頁面底部的 **「儲存」** 按鈕。

## 四、師資管理

點選左側 **「師資管理」**，管理分校的老師資料。

### 師資列表

頁面顯示所有老師的卡片，每張卡片包含：
- 老師姓名與照片
- 電話
- 認證狀態（質數教室認證老師）
- 薪資設定摘要

---

### 4-1 新增老師

點擊右上角 **「＋ 新增老師」** 按鈕：

| 欄位 | 說明 | 必填 |
|------|------|------|
| 老師姓名 | 真實姓名，會顯示在老師端和家長端 | 是 |
| 電話 | 老師聯絡電話 | 否 |
| 薪資方式 | 固定 / 時薪 / 百分比 | 是 |
| 薪資金額 | 依薪資方式填入對應金額 | 是 |

**薪資方式說明：**

| 類型 | 說明 | 範例 |
|------|------|------|
| 固定（Fixed） | 每堂固定金額 | 每堂 250 元 |
| 時薪（Hourly） | 按每小時計算 | 每小時 200 元 |
| 比例（Percentage） | 按課程收入的百分比 | 收入的 40% |

---

### 4-2 建立老師登入帳號

新增老師後，需要幫老師建立系統登入帳號：

1. 找到該老師的卡片，點擊 **「建立帳號」** 按鈕（🔑 圖示）
2. 系統自動產生帳號格式：**姓名@prime**
   - 例如：陳志明的帳號為 `陳志明@prime`
3. 設定初始密碼（至少 6 個字元）
4. 點擊確認建立

> 💡 **重要**：帳號格式固定為「姓名@prime」，務必將帳號和密碼交給老師，老師首次登入時須自行更改密碼。

---

### 4-3 編輯老師資料

點擊老師卡片的 **「✏️ 編輯」** 按鈕，可修改：
- 姓名、電話
- 自我介紹
- 薪資方式與金額
- 停用/啟用帳號

---

## 五、時段管理（排班）

點選左側 **「時段管理」**，管理分校的上課時段。

### 5-1 建立單一時段

點擊 **「＋ 新增時段」** 按鈕：

| 欄位 | 說明 |
|------|------|
| 日期 | 選擇上課日期（可點擊日期選擇器）|
| 開始時間 | 上課開始時間（例如 14:00） |
| 結束時間 | 上課結束時間（例如 16:00） |
| 指派老師 | 從下拉選單選擇負責老師 |
| 教室 | 選擇使用的教室 |
| 人數上限 | 最多幾位學生可預約此時段 |

填寫完成後，點擊 **「新增」** 確認建立。

---

### 5-2 批次建立時段

若每週固定排班，使用「批次建立」可一次建立多個時段：

1. 點擊 **「批次建立」** 按鈕
2. 選擇要排班的 **星期**（可多選，例如每週一、三、五）
3. 設定 **起始日期** 與 **結束日期**
4. 設定上課時間、老師、教室、人數上限
5. 點擊 **「批次新增」** 確認

**特殊節日自動略過：**

> ✅ 批次建立時，系統會自動檢查「特殊節日」設定。若您已在「分校資訊」中設定某日為休息，批次建立會自動跳過那一天，不需要手動刪除。

---

### 5-3 刪除時段

- **單一刪除**：在時段旁點擊 🗑️ 圖示，確認後刪除
- 若該時段已有學生預約，系統會提示需先取消預約才能刪除

## 六、手動新增學生（臨時加課）

> **注意**：這是內部操作模式，由**分校主任**直接在後台為學生建立預約，無需學生家長自行在家長端操作。

適合用於：
- 新學生第一次來上課
- 家長不方便操作 APP
- 補課、臨時加課需求

### 操作步驟

1. 點選左側 **「預約管理」**
2. 找到要加課的時段（依日期或老師篩選）
3. 在時段旁點擊 **「臨時加課」**（或 **「＋ 新增學生」**）按鈕
4. 填入學生資料：

| 欄位 | 說明 | 必填 |
|------|------|------|
| 學生姓名 | 孩子的真實姓名 | 是 |
| 年級 | 從下拉選單選擇（一年級至六年級） | 是 |
| 學校 | 就讀學校名稱 | 否 |
| 備註 | 主任備註（如：試聽課、補課） | 否 |

5. 點擊 **「新增」** 確認

系統會立即建立預約記錄，老師端也會即時看到此學生出現在該時段的點名列表中。

> 💡 **提示**：臨時加課建立的學生資料屬於「臨時學生」，不需要家長帳號。若家長日後要透過 APP 預約，需要另外在家長端完成註冊。

---

## 七、預約管理

點選左側 **「預約管理」**，查看和管理所有預約紀錄。

### 篩選功能

| 篩選項目 | 說明 |
|---------|------|
| 日期 | 查看特定日期的預約 |
| 老師 | 查看特定老師的預約 |
| 狀態 | 篩選已確認、已到、未到、已完成等狀態 |

### 預約狀態說明

| 狀態 | 說明 |
|------|------|
| 已確認 | 學生已預約，等待上課 |
| 已到 | 老師已點名，學生到場 |
| 未到 | 老師已標記，學生缺席 |
| 已完成 | 課程結束，系統已結算 |
| 已取消 | 預約已取消（不扣點） |

### 主任可執行的操作

- **查看詳細資訊**：點擊預約記錄查看完整資料
- **取消預約**：若需取消，點擊「取消」並確認

## 八、老師端登入

老師使用與主任 **相同的登入網址**，但切換到「**老師**」頁籤。

### 步驟

1. 開啟瀏覽器，前往 `/franchise-login`
2. 點擊上方 **「老師」** 頁籤（切換到老師登入模式）
3. 輸入帳號（格式：**姓名@prime**，例如 `陳志明@prime`）
4. 輸入密碼（初次登入需更改密碼）
5. 點擊 **「登入管理系統」**

![老師端登入頁面](step1-dashboard.png)

> 💡 **帳號說明**：帳號格式固定為「姓名@prime」，由分校主任建立並告知老師。

## 十、學生點名

上課前 **15 分鐘** 起，點名功能自動開放。

每位學生旁邊顯示兩個按鈕：**「已到」**（綠色）和 **「未到」**（紅色）

### 10-1 標記「已到」

學生到場後：

1. 點擊該學生旁的 **「已到」** 按鈕
2. 按鈕變為綠色的 **「已到」** 標籤
3. 右下角出現「點名成功」提示

![待點名狀態 — 顯示「已到」和「未到」按鈕](step5-restored.png)

![點名成功 — 學生已標記為出席](step6-checked-in.png)

---

### 10-2 標記「未到」

學生沒有出現時：

1. 點擊 **「未到」** 按鈕
2. 按鈕變為紅色的 **「未到」** 標籤
3. 系統自動提示「已標記未到」
4. 標記為未到的學生 **不需要填寫聯絡簿**

![標記未到 — 顯示紅色「未到」標籤](step3-marked-absent.png)

---

### 10-3 取消點名（按錯時的修正）

不小心按錯時：

1. 點擊「已到」或「未到」標籤旁的 **✕ 取消** 按鈕
2. 系統彈出確認對話框
3. 點擊 **「確認取消」**，學生恢復為待點名狀態
4. 重新點擊正確的按鈕

![取消未到 — 確認對話框](step4-unmark-dialog.png)

> ⚠️ **注意**：課程結束後就無法再修改點名狀態，請在上課期間確認完成。

## 十一、填寫聯絡簿

完成點名後，為每位 **已到** 的學生填寫聯絡簿，讓家長即時了解上課情況。

### 操作步驟

1. 在時段卡片中點擊 **「填寫聯絡簿」** 按鈕
2. 系統開啟聯絡簿填寫表單
3. 依序填寫各欄位
4. 點擊 **「儲存」** 按鈕

![聯絡簿填寫表單](step7-contact-book-form.png)

![聯絡簿儲存成功](step8-contact-book-saved.png)

### 欄位說明

| 欄位 | 說明 | 必填 | 家長可見 |
|------|------|------|---------|
| 今日上課單元 | 今天教的內容主題 | ✅ 是 | ✅ 是 |
| 教學進度 | 課本章節或教材進度 | 否 | ✅ 是 |
| 回家作業 | 指派給學生的練習 | 否 | ✅ 是 |
| 下次考試 | 下次小考的範圍或時間 | 否 | ✅ 是 |
| 小考成績 / 滿分 | 本次小考分數 | 否 | ✅ 是 |
| 老師備註（給家長） | 想讓家長知道的事情 | 否 | ✅ 是 |
| 內部筆記 | 老師自己的教學紀錄 | 否 | ❌ 否（僅老師看見） |

> 💡 **提示**：「內部筆記」只有老師自己看得到，家長端不會顯示，適合記錄教學觀察或下次備課提醒。

## 十二、每日工作完成度

行事曆下方顯示 **今日工作進度卡片**：

| 進度條 | 完成條件 |
|--------|---------|
| 點名進度 | 所有時段的每位學生都已標記「已到」或「未到」 |
| 聯絡簿進度 | 所有「已到」的學生都已填寫聯絡簿 |

- 兩個進度都達到 **100%** → 顯示 🟢 **「今日工作已完成」**
- 尚未完成 → 顯示 🟡 **「尚有未完成的工作」**

> ⚠️ **重要**：完成所有點名與聯絡簿填寫後，該日才納入薪資計算。請務必在當天完成！

## 十三、教材庫

切換到老師工作介面上方的 **「教材庫」** 頁籤。

### 13-1 數學練習網站

預設子分頁「📁 教材庫」內嵌質數教室的數學練習網站：
- 涵蓋各年級的題目與練習資源
- 可直接在頁面內瀏覽，無需另開視窗
- 可配合上課單元引導學生練習對應題型

### 13-2 期中考歸檔

切換到「📋 期中考歸檔」子分頁，查閱歷次試卷（PDF）：

1. 點擊 **「📋 期中考歸檔」** 按鈕切換
2. 列表顯示所有已上傳的試卷
3. 點擊試卷名稱，PDF 在頁面內開啟
4. 可翻頁、縮放查看
5. 點擊右上角按鈕可收起 PDF 閱讀器

> 試卷由總部統一上傳，老師無法自行上傳。

## 附錄：常見問題

**Q：老師點名按鈕是灰色的，無法點擊？**
> 點名功能在 **上課前 15 分鐘** 才開放，請稍後再試。

**Q：不小心按錯「已到」或「未到」怎麼辦？**
> 點擊標籤旁的 ✕ 按鈕，確認取消後即可重新選擇。

**Q：標記「未到」的學生還需要填聯絡簿嗎？**
> 不需要。缺席學生不需填聯絡簿，不會影響每日工作完成度。

**Q：忘記填聯絡簿，可以事後補填嗎？**
> 可以補填，但請盡早完成，以便家長即時看到上課紀錄。

**Q：老師忘記密碼怎麼辦？**
> 主任可在後台「師資管理」找到該老師，點擊「重設密碼」功能重新設定。

**Q：主任忘記密碼怎麼辦？**
> 請聯繫總部，由總部協助重設。

**Q：臨時加課的學生，家長可以在 APP 上看到上課紀錄嗎？**
> 臨時加課建立的是「臨時學生」紀錄，家長需要先在家長端完成帳號註冊，才能在 APP 看到孩子的上課記錄。

**Q：AI 自動生成介紹產生的文案可以修改嗎？**
> 可以！AI 生成後會填入欄位，您可以直接在欄位內修改內容，調整為最符合分校特色的版本後再儲存。
