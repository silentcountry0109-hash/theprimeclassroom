## 一、登入系統

1. 開啟瀏覽器，進入質數教室的分校登入頁面
2. 在上方切換到 **「老師」** 分頁
3. 輸入您的帳號（格式：`姓名@prime`）和密碼
4. 點擊 **「登入管理系統」**

> 帳號和密碼由分校主任建立並提供給您。首次登入時，系統會要求您更改密碼。

![登入頁面](step1-dashboard.png)

## 二、行事曆與今日排課

登入後，您會看到 **行事曆** 頁面，這是老師系統的首頁。

- 上方顯示當月月曆，有排課的日期下方會有小圓點標記
- 點擊日期後，下方會列出該天所有的 **時段卡片**
- 每張卡片顯示：
  - 上課時間（例如 14:00 - 16:00）
  - 教室名稱（例如 質數教室 大安教室）
  - 已預約人數 / 上限人數
  - 預約學生的姓名、年級、學號

![行事曆與排課總覽](step2-student-list.png)

## 三、學生點名

上課前 **15 分鐘** 起，點名功能會自動開放，您可以為每位學生標記出席狀態。

每位學生旁邊會顯示兩個按鈕：**「已到」**（綠色）和 **「未到」**（紅色）。

### 3-1 標記已到

當學生到場後，點擊該學生旁的 **「已到」** 按鈕：

- 按鈕會變成綠色的 **「已到」** 標籤
- 右下角會出現提示：「點名成功 — 學生已標記為出席」
- 點名進度會自動更新

![待點名狀態 — 顯示「已到」和「未到」按鈕](step5-restored.png)

![點名成功 — 顯示綠色「已到」標籤](step6-checked-in.png)

### 3-2 標記未到

如果學生沒有到場，點擊 **「未到」** 按鈕：

- 按鈕會變成紅色的 **「未到」** 標籤
- 右下角會出現提示：「已標記未到 — 學生已標記為缺席」
- 標記未到的學生 **不需要填寫聯絡簿**，該時段仍然可以完成每日工作

![標記未到 — 顯示紅色「未到」標籤](step3-marked-absent.png)

### 3-3 取消點名 / 取消未到

如果不小心按錯了，可以取消：

1. 點擊「已到」或「未到」標籤旁邊的 **X 按鈕**
2. 系統會彈出確認對話框，詢問您是否確定取消
3. 點擊 **「確認取消」** 後，學生會恢復為待點名狀態

> 課程結束後就無法再修改點名狀態，請在上課期間確認無誤。

![取消未到 — 確認對話框](step4-unmark-dialog.png)

## 四、填寫聯絡簿

完成點名後，請為每位 **已到** 的學生填寫聯絡簿。

### 操作步驟

1. 在時段卡片中，點擊 **「填寫聯絡簿」** 按鈕
2. 系統會開啟聯絡簿填寫表單，顯示該時段的學生
3. 依序填寫各項欄位
4. 點擊 **「儲存」** 按鈕
5. 儲存成功後，家長就能在家長端即時查看上課紀錄

![聯絡簿填寫表單](step7-contact-book-form.png)

![聯絡簿儲存成功](step8-contact-book-saved.png)

### 聯絡簿欄位說明

| 欄位 | 說明 | 是否必填 | 家長可見 |
|------|------|----------|----------|
| 今日上課單元 | 今天教的內容主題 | 必填 | 是 |
| 教學進度 | 課本章節或教材進度 | 選填 | 是 |
| 回家作業 | 指派給學生的練習 | 選填 | 是 |
| 下次考試 | 下次小考的範圍或時間 | 選填 | 是 |
| 小考成績 / 滿分 | 本次小考分數（如有考試） | 選填 | 是 |
| 老師備註（給家長端） | 想讓家長知道的事情 | 選填 | 是 |
| 內部筆記 | 老師自己的教學紀錄 | 選填 | 否 |

> 「內部筆記」只有老師自己看得到，家長端不會顯示。適合記錄教學觀察或下次上課的提醒。

## 五、每日工作紀錄

在行事曆下方，您會看到一個 **每日工作進度卡片**，顯示兩個進度條：

| 項目 | 說明 |
|------|------|
| 點名進度 | 所有時段的學生都已標記「已到」或「未到」即算完成 |
| 聯絡簿進度 | 所有「已到」的學生都已填寫聯絡簿即算完成 |

- 當兩個進度都達到 100% 時，顯示 **「今日工作已完成」**（綠色）
- 尚未完成時，顯示 **「尚有未完成的工作」**（黃色）

> 完成所有點名與聯絡簿填寫後，該日才會計入薪資計算。請務必在當天完成。

## 六、預約狀態對照表

| 狀態 | 老師端顯示 | 家長端顯示 | 說明 |
|------|-----------|-----------|------|
| 已確認 | 顯示「已到」「未到」按鈕 | 已確認 | 家長已成功預約，等待上課 |
| 已到 | 綠色「已到」標籤 | 上課中 | 老師已確認學生出席 |
| 未到 | 紅色「未到」標籤 | 未到 | 老師已確認學生缺席 |
| 已完成 | 顯示於歷史紀錄 | 已完成 | 課程結束，系統自動結算 |
| 已取消 | 不顯示 | 已取消 | 家長在上課前取消預約 |

## 七、常見問題

**Q: 點名按鈕按不了，顯示灰色？**
> 點名功能會在上課前 15 分鐘自動開放。如果按鈕是灰色的，表示還沒到開放時間，請稍候。

**Q: 不小心按錯「已到」或「未到」怎麼辦？**
> 點擊標籤旁邊的 X 按鈕，確認取消後即可恢復為待點名狀態，重新選擇即可。

**Q: 學生沒來上課，需要填聯絡簿嗎？**
> 不需要。標記為「未到」的學生不需要填寫聯絡簿，不會影響每日工作完成度。

**Q: 下課後忘記填聯絡簿怎麼辦？**
> 聯絡簿可以在課程結束後補填。但建議盡早完成，這樣家長才能即時看到上課紀錄。

**Q: 忘記密碼怎麼辦？**
> 請聯繫您的分校主任，主任可以在後台為您重設密碼。

**Q: 可以在手機上操作嗎？**
> 可以。老師系統支援手機瀏覽器，操作方式和電腦版相同。
