## 一、家長端

### 入口

- 網址：`/parent-login`
- 登入後進入家長儀表板 `/parent-dashboard`

### 1. 登入與帳號

| 項目       | 說明                                         |
| ---------- | -------------------------------------------- |
| 帳號類型   | 帳號密碼登入（credential login）             |
| 角色       | `parent`                                     |
| 登出       | 右上角登出按鈕，清除 session 後導回登入頁    |

**操作流程：**

1. 進入 `/parent-login` 頁面
2. 輸入帳號與密碼
3. 系統驗證後建立 session，導向 `/parent-dashboard`
4. 若 session 過期或無效，自動導回登入頁

---

### 2. 首頁總覽（Overview）

首頁為家長登入後的預設頁面，提供即時概況。

**顯示內容：**

- **問候語**：依時間顯示「早安 / 午安 / 晚安」，附帶今日日期
- **待上課提醒**：右上角顯示「N 堂待上課」按鈕，點擊跳轉至預約紀錄
- **下一堂課卡片**：
  - 顯示最近一堂已確認課程的教室名稱、老師、時間、孩子名稱
  - 顯示倒數計時（N 天 N 小時後 / N 小時 N 分鐘後 / N 分鐘後）
  - 倒數每分鐘自動更新
  - 點擊整張卡片可跳轉至預約紀錄頁
- **孩子概況**：
  - 每位孩子顯示頭像、姓名、年級、學校
  - 顯示「待上課」與「已完成」堂數
  - 若尚無孩子，顯示引導按鈕「登記孩子」
- **快捷入口**（三宮格）：
  - 預約課程 → 跳轉至預約課程頁籤
  - N 堂完成 → 跳轉至預約紀錄
  - 教材商城 → 跳轉至商城頁籤
- **即將到來列表**：
  - 除了下一堂課外，顯示接下來 2~3 堂即將上課的預約
  - 包含日期、時間、教室、老師、孩子資訊
- **空狀態**：若有孩子但無預約，顯示引導「搜尋教室」按鈕

### 3. 預約課程（Book）

預約課程為三步驟流程，頂部有步驟指示器（StepIndicator）。

#### 步驟一：搜尋教室

**操作流程：**

1. 選擇**縣市**（下拉選單，台灣各縣市）
2. 選擇**行政區**（依縣市動態載入對應行政區）
3. 系統自動搜尋該地區的分校教室
4. 顯示搜尋結果列表

**搜尋結果卡片資訊：**

- 封面照片（若有設定封面圖或教室照片）
- 教室名稱
- 地址
- 電話
- 老師數量
- 可預約時段數
- 評分（4.5 分以上顯示）

5. 點擊任一教室卡片進入步驟二

#### 步驟二：選擇時段

**顯示內容：**

- 教室基本資訊（名稱、地址、電話、老師數）
- 教室介紹文字
- 教室照片輪播（自動輪播，3.5 秒間隔，可手動切換，滑鼠懸停暫停）
- 師資介紹卡片列表：
  - 老師姓名、認證標記、評分、專長標籤、個人簡介
- 可預約時段列表：
  - 依日期分頁籤（日期 + 星期幾）
  - 每個時段卡片顯示：時間區間、座位使用指示燈（圓點）、剩餘名額
  - 時段內的老師資訊（姓名、認證、評分、專長）
  - 「預約」按鈕（剩餘名額 > 0 才可點擊）

**操作流程：**

1. 切換日期頁籤選擇日期
2. 點擊可用時段的「預約」按鈕
3. 若尚未登記孩子，會提示「請先登記孩子」
4. 進入步驟三

#### 步驟三：確認預約

**顯示內容：**

- 教室名稱與地址、電話
- 預約日期、時間、剩餘名額
- 授課老師資訊（含認證、評分、專長）
- 選擇上課孩子（單一孩子自動選取，多位孩子需手動選擇）
- 取消政策提醒（黃底文字）：「課程開始前 4 小時內取消，系統仍會扣除點數計費」

**操作流程：**

1. 確認教室與時段資訊
2. 選擇上課的孩子
3. 點擊「確認預約」
4. 預約成功後：
   - 顯示成功提示
   - **自動檢查週期性排課**：系統查詢同一時段在未來 4 週的可用時段
   - 若有可用週期時段，彈出「週期性排課」對話框

#### 週期性排課（Recurring Booking）

預約成功後自動觸發，非必要步驟。

**顯示內容：**

- 標題：「每週X {時間}，是否同時預約未來幾週？」
- 未來 4 週同一時段的可用情況列表：
  - 可預約：顯示剩餘名額，預設勾選
  - 已預約：灰色不可選
  - 未開課：灰色不可選
  - 已額滿：灰色不可選

**操作流程：**

1. 勾選或取消勾選要預約的週次
2. 點擊「預約 N 週」批次預約
3. 或點擊「不用了」跳過

### 4. 我的孩子（Children）

管理孩子的學習檔案。

#### 孩子列表

**每位孩子卡片顯示：**

- 頭像（依性別自動切換男生/女生頭像）
- 姓名、年級標籤
- 學校名稱（若有填寫）
- 待上課堂數、已完成堂數
- 學習進度條（已完成 / 總堂數 百分比）
- 下一堂課資訊（日期、時間、教室名稱）
- 編輯按鈕（鉛筆圖示）

#### 新增孩子

**操作流程：**

1. 點擊「新增孩子」按鈕
2. 填寫表單：
   - **姓名**（必填）
   - **性別**：點選男生或女生頭像按鈕
   - **年級**（必填）：1~6 年級下拉選單
   - **學校**（選填）
3. 點擊「確認新增」

#### 編輯孩子

**操作流程：**

1. 點擊孩子卡片上的鉛筆圖示
2. 表單自動帶入現有資料
3. 修改後點擊「儲存變更」

**注意：孩子不可刪除**，以保留完整的學習紀錄。

---

### 5. 預約紀錄（Bookings）

查看與管理所有預約課程。

#### 篩選功能

**時間區間篩選：**

- 開始日期與結束日期（date picker）
- 快速區間按鈕：7 天 / 14 天 / 30 天 / 90 天
- 預設顯示今天起算 14 天內

**孩子篩選**（多位孩子時顯示）：

- 每位孩子一張摘要卡片，顯示頭像、姓名、待上課數、已完成數、下一堂課
- 點擊孩子卡片篩選該孩子的預約
- 再次點擊取消篩選
- 「全部孩子」按鈕可清除篩選

**狀態篩選：**

- 全部 / 即將上課 / 已完成 / 已取消
- 每個篩選按鈕顯示該狀態的數量

#### 預約列表

- 依日期分組顯示，每組顯示日期標籤（今天/明天/月/日+星期幾）與堂數
- 排序規則：已確認 > 其他狀態；已確認按時間正序，其他按時間倒序
- 每筆預約卡片顯示：
  - 上課時間（開始~結束）
  - 教室名稱
  - 預約狀態標籤（已確認：Tiffany 色 / 已完成：綠色 / 已取消：灰色）
  - 老師姓名
  - 孩子名稱與頭像（多位孩子且未篩選時顯示）
  - 取消按鈕（僅「已確認」狀態顯示）

#### 行事曆同步

- 右上角「加入行事曆」按鈕
- 下載 `.ics` 檔案，包含所有已確認課程
- 每堂課設有開始前 6 小時提醒

#### 取消預約

**操作流程：**

1. 點擊預約卡片上的 X 按鈕
2. 彈出取消對話框，顯示：
   - 預約資訊（教室、日期、時間、孩子）
   - 紅色提醒：「課程開始前 4 小時內取消，系統仍會扣除點數計費」
3. **滑動解鎖機制**：
   - 拖曳滑桿至最右側才能啟用「確認取消」按鈕
   - 滑動過程有 emoji 表情變化（😊→🤔→😮→😰→😢→✅）
   - 未滑到底放開會自動彈回
4. 解鎖後點擊「確認取消」
5. **4 小時規則**：若距離課程開始不足 4 小時，伺服器拒絕取消並回傳錯誤訊息

#### 底部提醒

頁面底部固定顯示紅色取消政策提醒框。

### 6. 教材商城（Shop）

為孩子選購教材與教具。

#### 檢視切換

頂部有「商品 / 訂單」切換按鈕。

#### 商品瀏覽

**分類篩選：**

- 全部 / 教材 / 教具

**商品卡片：**

- 商品圖片（無圖片顯示漸層佔位圖）
- 商品名稱、分類標籤
- 商品描述（最多兩行）
- 價格顯示：
  - 有特價：紅色特價 + 刪除線原價
  - 無特價：標準價格
- 庫存警示：剩餘 5 件以下顯示「僅剩 N 件」
- 「加入」按鈕（缺貨時顯示「缺貨」且不可點擊）

**操作流程：**

1. 選擇分類篩選
2. 點擊商品的「加入」按鈕
3. 商品加入購物車，顯示成功提示
4. 重複加入同商品會自動合併數量

#### 購物車

- 右上角購物車圖示，顯示商品件數紅色徽章
- 點擊開啟右側滑出面板

**購物車內容：**

- 商品圖片縮圖、名稱、單價
- 數量控制：+ / - 按鈕
  - 數量減至 0 自動移除
- 刪除按鈕（垃圾桶圖示）
- 小計金額
- 「結帳」按鈕

#### 結帳

**操作流程：**

1. 在購物車點擊「結帳」
2. 彈出確認訂單對話框：
   - 商品清單（名稱 x 數量 = 金額）
   - 總計金額
   - 備註欄位（選填）
3. 點擊「確認下單」
4. 訂單建立成功，購物車清空，顯示成功提示

#### 訂單紀錄

切換至「訂單」檢視。

**訂單卡片：**

- 訂單編號、下單日期時間
- 訂單總金額
- 訂單狀態標籤：
  - 待處理（黃色）
  - 已付款（藍色）
  - 已出貨（紫色）
  - 已完成（綠色）
  - 已取消（灰色）
- 點擊展開/收合訂單明細：
  - 各商品名稱、數量、金額
  - 備註（若有）

## 二、分校管理後台端

### 入口

- 網址：`/franchise-login`
- 登入後進入分校管理儀表板 `/franchise-admin`

### 1. 登入與帳號

| 項目       | 說明                                         |
| ---------- | -------------------------------------------- |
| 帳號類型   | 帳號密碼登入（credential login）             |
| 角色       | `franchise_admin`                            |
| 登出       | 側邊欄底部登出按鈕                           |

**帳號對應關係：** 每個分校帳號綁定一間分校（franchiseId），登入後只能查看和管理該分校的資料。

---

### 2. 分校總覽（Overview）

#### 即時統計卡片

顯示分校的即時數據：

| 卡片     | 資料來源                       |
| -------- | ------------------------------ |
| 老師人數 | 該分校的老師總數               |
| 可用時段 | 該分校的時段總數               |
| 總預約數 | 該分校所有預約筆數             |
| 已確認   | 狀態為 confirmed 的預約筆數    |

#### 統計分析區（依時間區間）

**時間區間選擇：**

- 快速預設按鈕：近 7 天 / 近 14 天 / 近 1 個月 / 近 3 個月
- 自訂開始日期 / 結束日期（date picker）
- 切換預設會自動更新日期範圍
- 手動修改日期會清除預設選取狀態

**區間統計卡片：**

| 卡片       | 內容                                     |
| ---------- | ---------------------------------------- |
| 開課時段   | 時段數量 + 座位總數                      |
| 預約數     | 預約總數 + 已確認數                      |
| 取消數     | 取消預約數 + 取消率（取消數/總預約數）   |
| 座位使用率 | 百分比 + 已使用/總座位數                 |

#### 每日座位使用圖表

- 橫向長條圖，每天一列
- 顯示日期（月/日 + 星期幾）
- 長條顏色：
  - Tiffany 色：使用率 ≥ 80%
  - 珊瑚色：使用率 50-79%
  - 灰色：使用率 < 50%
- 右側顯示「已用/總座位 (使用率%）」
- 底部圖例說明
- 最大高度限制，超出可捲動

#### 老師授課統計表

依所選時間區間統計每位老師的授課數據。

**表格欄位：**

| 欄位     | 說明                                                       |
| -------- | ---------------------------------------------------------- |
| 老師     | 老師姓名                                                   |
| 時段數   | 該老師在區間內被指派的時段數                               |
| 預約數   | 該老師時段內的總預約筆數                                   |
| 已確認   | 狀態為 confirmed 的預約數（Tiffany 色）                    |
| 已完成   | 狀態為 completed 的預約數（綠色）                          |
| 已取消   | 狀態為 cancelled 的預約數（紅色，為 0 時灰色）             |
| 取消率   | 取消數/總預約數，顏色標示：≥30% 紅色、≥15% 黃色、其他灰色 |
| 已用座位 | 該老師時段的已預約座位數                                   |
| 使用率   | 已用座位/(時段數×5)，顏色標示：≥80% Tiffany、≥50% 珊瑚    |

### 3. 分校資訊（Info）

查看與編輯分校的基本資訊。

#### 檢視模式

顯示以下資訊（唯讀）：

- 分校名稱
- 地址（縣市+行政區+詳細地址）
- 電話
- 分校介紹
- 標籤（如：家長好評推薦、年度績優校區等）
- 鄰近學校
- 評分與評價數

#### 編輯模式

**操作流程：**

1. 點擊「編輯資訊」按鈕
2. 可編輯欄位：
   - **電話**：文字輸入
   - **分校介紹**：多行文字輸入
   - **標籤**：從預設選項新增（家長好評推薦、年度績優校區、成績進步快速、新開幕、小班教學），可逐一刪除
   - **鄰近學校**：自由輸入學校名稱，Enter 或點「加入」新增，可逐一刪除
3. 點擊「儲存變更」或「取消」

**注意：** 分校名稱、地址、評分為系統管理欄位，不可由分校端修改。

#### 教室照片管理

分校資訊頁面下方的獨立區塊。

**功能：**

- **上傳照片**：支援 JPG、PNG、GIF、WebP，最大 5MB
- **設定封面照片**：hover 照片後點擊星號圖示
  - 封面照片會顯示在家長搜尋結果的教室卡片頂部
  - 已設為封面的照片會有 Tiffany 色邊框和「封面」標籤
  - 再次點擊星號可取消封面
- **刪除照片**：hover 照片後點擊垃圾桶圖示
- 照片以 2~3 欄網格排列

---

### 4. 師資管理（Coaches）

管理分校的老師資料。

#### 老師列表

每位老師卡片顯示：

- 姓名首字圓形頭像
- 姓名、認證標記
- 專長標籤
- 評分（有評分時顯示）
- 編輯 / 刪除按鈕

#### 新增老師

**操作流程：**

1. 點擊「新增老師」按鈕
2. 填寫表單：
   - **姓名**（必填）
   - **簡介**（選填，多行文字）
   - **專長**：自由輸入，可新增多個，可逐一刪除
   - **認證狀態**：開關切換（預設已認證）
3. 點擊「新增」

#### 編輯老師

1. 點擊老師卡片上的編輯按鈕
2. 表單帶入現有資料
3. 修改後點擊「更新」

#### 刪除老師

1. 點擊刪除按鈕
2. 確認對話框確認後刪除
3. 該老師已指派的時段會自動解除指派（coachId 設為 null）

---

### 5. 時段管理（Time Slots）

管理分校的可預約時段。

#### 時段列表

- 按日期+時間排序
- 每個時段顯示：日期（含星期幾）、時間區間、指派老師、已預約/總座位數
- 刪除按鈕（確認後刪除）

#### 新增時段

**操作流程：**

1. 點擊「新增時段」按鈕
2. 填寫表單：
   - **日期**（必填，日期選擇器）
   - **開始時間**（必填，時間選擇器）
   - **結束時間**（必填，時間選擇器）
   - **指派老師**（選填，下拉選單列出該分校所有老師）
   - **座位數**（預設 5，數字輸入）
3. 點擊「新增時段」

**注意：**

- 時段建立後不可編輯，只能刪除重建
- 時段一旦有預約，刪除前需確認
- 家長在預約課程中看到的就是這些時段

### 6. 預約管理（Bookings）

查看該分校收到的所有預約。

#### 預約列表

每筆預約顯示：

- 學生姓名、年級、學校
- 預約日期（含星期幾）、時間區間
- 預約狀態標籤：
  - 已確認（Tiffany 色）
  - 已取消（灰色）
  - 已完成（黃色）
- 預約建立日期

**注意：** 分校端目前為唯讀檢視，不可直接修改預約狀態。預約狀態變更由家長端操作（取消）或系統自動處理（完成）。

## 三、業務規則

### 預約取消政策

| 規則             | 說明                                         |
| ---------------- | -------------------------------------------- |
| 取消期限         | 課程開始前 4 小時                            |
| 逾時取消         | 系統拒絕操作，回傳錯誤訊息                   |
| 逾時取消後果     | 扣除點數計費（即使課程未實際上課）           |
| 提醒位置         | 預約確認步驟、取消對話框、預約紀錄底部       |

### 週期性排課

| 規則             | 說明                                         |
| ---------------- | -------------------------------------------- |
| 觸發時機         | 單次預約成功後自動檢查                       |
| 查詢範圍         | 未來 4 週同一星期幾、同一時段                |
| 預設行為         | 所有可用週次預設勾選                         |
| 跳過選項         | 可點擊「不用了」跳過                         |
| 批次結果         | 顯示成功/失敗數量                            |

### 孩子管理

| 規則             | 說明                                         |
| ---------------- | -------------------------------------------- |
| 新增             | 需填寫姓名、年級（必填），性別、學校（選填） |
| 編輯             | 可修改所有欄位                               |
| 刪除             | 不可刪除，以保留學習紀錄                     |
| 性別預設         | 男生（male）                                 |

### 商城訂單

| 規則             | 說明                                         |
| ---------------- | -------------------------------------------- |
| 價格單位         | 新台幣（TWD），整數儲存                      |
| 購物車合併       | 加入已在購物車的商品會自動增加數量           |
| 訂單狀態流程     | 待處理 → 已付款 → 已出貨 → 已完成           |
| 取消訂單         | 由 HQ 管理後台操作                           |
| 庫存檢查         | 缺貨商品無法加入購物車                       |

### 分校照片管理

| 規則             | 說明                                         |
| ---------------- | -------------------------------------------- |
| 格式支援         | JPG、PNG、GIF、WebP                          |
| 大小限制         | 單張最大 5MB                                 |
| 封面照片         | 每間分校僅一張封面，顯示在家長搜尋結果       |
| 存放路徑         | 上傳至 `uploads/` 目錄                       |

### 老師授課統計

| 規則             | 說明                                         |
| ---------------- | -------------------------------------------- |
| 統計範圍         | 依選擇的時間區間計算                         |
| 使用率計算       | 已用座位 ÷（時段數 × 5）                    |
| 取消率計算       | 已取消預約 ÷ 總預約數                       |
| 取消率警示       | ≥30% 紅色、≥15% 黃色                        |
